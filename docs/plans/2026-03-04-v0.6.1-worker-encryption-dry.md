# v0.6.1 — Worker Encryption, DIFAT Docs, DRY Improvements

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add password/encryption support to the Worker API, document the OLE2 DIFAT limitation, and deduplicate shared constants and crypto helpers.

**Architecture:** The Worker API already has `readBuffer`/`writeBuffer` — we extend both with optional `password` parameter. The WASM bridge already exports `readWithPassword`/`writeWithPassword`. On the Rust side, we consolidate `OLE2_MAGIC` into `ole2/mod.rs`, extract a generic `with_hash_alg` dispatcher to eliminate triplicated match blocks, and extract a `compute_hmac` helper to deduplicate HMAC computation.

**Tech Stack:** TypeScript (Vitest), Rust (sha2, hmac, wasm-bindgen)

// Hashing helpers
// ---------------------------------------------------------------------------

/// Dispatch a closure over the correct hash algorithm type.
/// The closure receives a freshly constructed `Digest` instance.
fn with_hash_alg<F, R>(alg: &str, f: F) -> Result<R>
where
    F: FnOnce(&mut dyn digest::DynDigest) -> R,
{
    match alg {
        "SHA512" => Ok(f(&mut Sha512::new())),
        "SHA256" => Ok(f(&mut Sha256::new())),
        "SHA1" => Ok(f(&mut Sha1::new())),
        _ => Err(ModernXlsxError::PasswordProtected(format!(
            "Unsupported hash algorithm: {alg}"
        ))),
    }
}

/// Hash bytes with the named algorithm.
fn hash_bytes(alg: &str, data: &[u8]) -> Result<Vec<u8>> {
    with_hash_alg(alg, |h| {
        h.update(data);
        h.finalize_reset().to_vec()
    })
}

/// Hash the concatenation of two byte slices: H(prefix || data).
fn hash_bytes_with_prefix(prefix: &[u8], data: &[u8], alg: &str) -> Result<Vec<u8>> {
    with_hash_alg(alg, |h| {
        h.update(prefix);
        h.update(data);
        h.finalize_reset().to_vec()
    })
}
```

/// Compute HMAC of `data` using the named hash algorithm and `key`.
fn compute_hmac(alg: &str, key: &[u8], data: &[u8]) -> Result<Vec<u8>> {
    match alg {
        "SHA512" => {
            let mut mac = Hmac::<Sha512>::new_from_slice(key)
                .map_err(|e| ModernXlsxError::PasswordProtected(format!("HMAC init: {e}")))?;
            mac.update(data);
            Ok(mac.finalize().into_bytes().to_vec())
        }
        "SHA256" => {
            let mut mac = Hmac::<Sha256>::new_from_slice(key)
                .map_err(|e| ModernXlsxError::PasswordProtected(format!("HMAC init: {e}")))?;
            mac.update(data);
            Ok(mac.finalize().into_bytes().to_vec())
        }
        _ => Err(ModernXlsxError::PasswordProtected(format!(
            "Unsupported hash algorithm for HMAC: {alg}"
        ))),
    }
}
```
